// cors note
CORS stands for Cross-Origin Resource Sharing. It is a browser security feature that restricts cross-origin HTTP requests
 with other servers and specifies which domains can access your resources1. CORS allows you to configure the web API’s security 
 and prevent unknown websites from accessing your server’s resources2.
